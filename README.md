# 🚀 Scroll-Driven Hero Section Animation

This project is a modern, animated hero section built using React, Tailwind CSS, and GSAP. It demonstrates smooth scroll-based animations and interactive UI behavior inspired by real-world landing pages.

## 🔥 Features

- ✨ Letter-by-letter animated headline on page load  
- 📊 Animated statistics with smooth stagger effects  
- 🚗 Scroll-driven car animation using GSAP ScrollTrigger  
- 🎯 Smooth and responsive animations tied to scroll position  
- ⚡ High performance using transform-based animations  

## 🛠 Tech Stack

- React.js  
- Tailwind CSS  
- GSAP (GreenSock Animation Platform)  
- ScrollTrigger  

## 🎬 How It Works

- On page load, the headline animates with a smooth staggered effect  
- Statistics appear one by one with a subtle delay  
- As the user scrolls, the car moves smoothly across the screen  
- The animation is synced with scroll using ScrollTrigger (`scrub`)
